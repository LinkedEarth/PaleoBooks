

Dependencies
============

The notebooks gathered here use the standard Python stack, namely

* `NumPy`_
* `Pandas`_
* `SciPy`_
* `scikit-learn`_
* `Matplotlib`_
* `seaborn`_

In addition, it leverages packages specific to the geosciences, like

* `Xarray`_
* `Cartopy`_
* `Pyleoclim`_
* `PyLipd`_
